# Executive Intelligence Briefing (EIB)
## Architecture

Version: Foundation (v0.1)

---

# Mission

Create a reusable Executive Intelligence system that enables individuals to capture knowledge, organize experience, improve decision making, and leave behind a better starting point for the next person.

---

# Guiding Principle

> Every solution should leave behind a better starting point for the next person.

---

# Repository Strategy

The EIB consists of two complementary repositories.

## Public Repository

Purpose

Provide reusable templates, documentation, prompts, examples, and implementation guidance that anyone can use.

Contains

- Templates
- Setup Guides
- Prompt Library
- Examples
- Documentation
- Roadmaps

Never contains personal information.

---

## Private Repository

Purpose

Capture personal knowledge and information.

Contains

- Health
- Retirement
- Financial
- Family
- Work
- Projects
- Reports

May contain confidential or sensitive information.

---

# Relationship

Public provides reusable structure.

Private provides personalized implementation.

Every reusable improvement discovered within the Private repository should be evaluated for promotion into the Public repository.

---

# Design Philosophy

Templates before personalization.

Reasoning before decisions.

Simple before clever.

Public when practical.

Private when necessary.

Version everything.

---

# Success

The EIB succeeds when every improvement makes it easier for the next person to begin.
